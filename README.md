# 2026-pop

# 📊 연령별 인구 구조 대시보드

행정안전부 주민등록 연령별 인구현황 데이터를 활용해, 지역별 인구 구조를 한눈에 보고
인구 구조가 가장 비슷한 지역을 찾아주는 Streamlit 웹앱입니다.

## 주요 기능

- **지역별 연령 인구 구조 시각화**: 시/도, 시/군/구, 읍/면/동 단위로 선택 가능하며,
  선택한 지역의 연령대별 인구수를 계/남/여 꺾은선 그래프로 보여줍니다.
- **지역 검색 + 선택**: 사이드바에서 지역명을 검색어로 필터링한 뒤, 드롭다운에서 선택합니다.
- **인구구조 유사 지역 Top 5**: 선택한 지역과 연령대별 인구 *비율*(절대 인구수가 아님)이
  코사인 유사도 기준으로 가장 비슷한 전국 지역 5곳을 찾아 그래프로 비교합니다.
- 모든 그래프는 [Plotly](https://plotly.com/python/)로 제작되어 확대/축소, 값 hover 등
  인터랙티브 조작이 가능합니다.

## 데이터

- 출처: 행정안전부 주민등록 인구통계 (연령별 인구현황, 월간)
- 파일: `202606_202606_연령별인구현황_월간_2.csv` (2026년 6월 기준)
- 인코딩: `CP949(EUC-KR)`

> ⚠️ 데이터 파일은 `app.py`(또는 `main.py`)와 **같은 폴더**에 있어야 합니다.
> 이 저장소를 통째로 클론/포크하면 별도 설정 없이 바로 실행됩니다.

## 폴더 구조

```
.
├── app.py              # (또는 main.py) Streamlit 앱 본체
├── requirements.txt     # 의존 패키지 목록
└── 202606_202606_연령별인구현황_월간_2.csv   # 원본 데이터
```

## 로컬에서 실행하기

```bash
git clone <이 저장소 URL>
cd <저장소 폴더>
pip install -r requirements.txt
streamlit run app.py
```

브라우저에서 `http://localhost:8501` 로 접속하면 앱이 열립니다.

## Streamlit Cloud 배포하기

1. 이 저장소를 GitHub에 그대로 push합니다 (`app.py`, `requirements.txt`, CSV 파일 모두 같은 폴더).
2. [share.streamlit.io](https://share.streamlit.io) 에서 **New app** → 저장소 선택.
3. **Main file path**를 실제 앱 파일명(`app.py` 또는 `main.py`)에 맞게 지정합니다.
4. Deploy를 누르면 자동으로 `requirements.txt`의 패키지를 설치하고 앱을 실행합니다.

배포 후 앱이 CSV 파일을 못 찾는다는 에러가 나면:
- 데이터 파일이 실제로 GitHub 저장소에 커밋/푸시되었는지 확인
- 파일명이 코드의 `DATA_FILENAME` 값과 정확히 일치하는지 확인
- Streamlit Cloud 관리 화면(Manage app)에서 **Reboot app**으로 재배포

## 유사도 계산 방식 (참고)

지역 간 인구구조 유사도는 각 지역의 연령대별(0~100세) 인구 **비율**을 벡터로 만들어
코사인 유사도로 비교합니다. 절대 인구수가 아니라 연령 분포의 "모양"을 비교하는 지표이므로,
인구 규모가 크게 다른 지역끼리도 연령 구조가 비슷하면 높은 유사도로 나타날 수 있습니다.
또한 원본 데이터에는 시/도·시/군/구·읍/면/동이 함께 섞여 있어, 서로 다른 행정 단위의
지역이 결과에 함께 나타날 수 있습니다.
